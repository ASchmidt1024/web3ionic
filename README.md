# Ionic Web3

Ionic starter app for developing dapps with web3.js

## Usage

Keep shure that Ionic is installed.

```
npm install -g ionic
```

Install the starter app.

```
npm install
```

Run the server.

```
ionic serve
```

A browser should be opened automatically.
